Q1.HTMLとは何ですか？
A1.ウェブサイトを記述するための言語です。 インターネットのサイトは、HTMLを使って文章のように記述されています。 ブラウザは、HTMLで書かれた文章（ソース）を読み込むことで、その内容に従って、サイトのホームページを表示しています
Q2.CSSとは何ですか？
A2.CSSとはスタイルシートのことです。スタイルシートとは、一言でいうなら「Webページのレイアウトを定義する技術」です。
Q1.BootStrapとは何ですか？
1.HTML/CSS/JavaScriptから構成されるCSSフレームワークです。Bootstrapの⽬的として、何度も同じCSSを書かないことが挙げられます。つまり、Bootstrapでは、webページでよく使われるボタンやフォーム、flexboxなどのレイアウトがあらかじめ保存されているので、CSSを書かずともHTML側で呼び出すことができ、ある程度の品質のレイアウトを作成することができます。Bootstrapを使うと開発を⾼速化することができるので、コードを書くこと⾃体にあまり時間をかけたくない⽅におすすめです。
Q2.JavaScriptとは何ですか？
A2.JavaScriptは「スクリプト言語」と呼ばれる類のプログラミング言語です。 スクリプト言語が分からない人は、ちょっと間違った説明ですが「書くのが簡単なプログラミング言語」と解釈してください。 書いて動かすまでの手間が、普通のプログラミング言語に比べて少なくて済みます。
